# coroutines-workshop-cpponsea2022

Coroutines workshop exercise code for the conference C++ on Sea 2022.

All exercise code is left as-is, solution code is fully refactored and styled.

## Building

Requirements:

* GCC v12 or higher,
* CMake v3.15 or higher.

Instructions:

```sh
mkdir build
ccmake -S . -B ./build/
cd build
make
```
